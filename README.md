# avtomat
Тип: Мили
Пример: Движение игрока в игре


    ## 1. Компоненты автомата
    
    **Множество состояний** $Q = \{q_0, q_1, q_2, q_3, q_4\}$:
    * $q_0$ — состояние покоя (игрок стоит на месте);
    * $q_1$ — движение влево;
    * $q_2$ — движение вправо;
    * $q_3$ — прыжок;
    * $q_4$ — падение (после прыжка, пока не коснулся земли).
    
    **Входной алфавит** $\Sigma = \{\text{влево}, \text{вправо}, \text{прыжок}, \text{стоп}, \text{земля}\}$:
    * «влево» — нажатие клавиши ←;
    * «вправо» — нажатие клавиши →;
    * «прыжок» — нажатие клавиши пробела;
    * «стоп» — отпускание всех клавиш;
    * «земля» — сигнал от движка игры, что игрок коснулся поверхности.
    
    **Выходной алфавит** $\Gamma = \{\text{stop}, \text{move\_left}, \text{move\_right}, \text{jump}, \text{fall}\}$:
    * команды, отправляемые игровому движку для анимации и физики.
    
    **Функция переходов** $\delta: Q \times \Sigma \to Q$:
    * $\delta(q_0, \text{влево}) = q_1$;
    * $\delta(q_0, \text{вправо}) = q_2$;
    * $\delta(q_0, \text{прыжок}) = q_3$;
    * $\delta(q_0, \text{стоп}) = q_0$;
    * $\delta(q_1, \text{стоп}) = q_0$;
    * $\delta(q_1, \text{прыжок}) = q_3$;
    * $\delta(q_2, \text{стоп}) = q_0$;
    * $\delta(q_2, \text{прыжок}) = q_3$;
    * $\delta(q_3, \text{земля}) = q_0$;
    * $\delta(q_3, \text{влево}) = q_4$;
    * $\delta(q_3, \text{вправо}) = q_4$;
    * $\delta(q_4, \text{земля}) = q_0$.
    
    **Начальное состояние** $q_0$ (игрок начинает игру стоя на земле).
    
    **Конечные состояния** отсутствуют — это автомат непрерывного действия, он работает на протяжении всей игры.

    ## 4. Проверка работы автомата
    
    
    **Сценарий 1. Простое движение влево и остановка**
    * Вход: `влево → стоп`
    * Последовательность:
        * $q_0 \xrightarrow{\text{влево/move\_left}} q_1$
        * $q_1 \xrightarrow{\text{стоп/stop}} q_0$
    * Выход: `move_left → stop`
    * Результат: игрок двинулся влево и остановился.
    
    **Сценарий 2. Прыжок во время движения вправо**
    * Вход: `вправо → прыжок → земля`
    * Последовательность:
        * $q_0 \xrightarrow{\text{вправо/move\_right}} q_2$
        * $q_2 \xrightarrow{\text{прыжок/jump}} q_3$
        * $q_3 \xrightarrow{\text{земля/stop}} q_0$
    * Выход: `move_right → jump → stop`
    * Результат: игрок бежал вправо, прыгнул, приземлился и остановился.
    
    **Сценарий 3. Падение с платформы**
    * Вход: `прыжок → влево → земля`
    * Последовательность:
        * $q_0 \xrightarrow{\text{прыжок/jump}} q_3$
        * $q_3 \xrightarrow{\text{влево/fall}} q_4$
        * $q_4 \xrightarrow{\text{земля/stop}} q_0$
    * Выход: `jump → fall → stop`
    * Результат: игрок прыгнул, начал падать влево, приземлился.
    
    **Сценарий 4. Бездействие**
    * Вход: `стоп`
    * Последовательность: $q_0 \xrightarrow{\text{стоп/stop}} q_0$
    * Выход: `stop`
    * Результат: игрок остаётся на месте.
